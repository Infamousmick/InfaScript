## How to run...? 

#### 01. Download [Setup.sh](setup.sh)
#### 02. Move "setup.sh" file to a folder of your chooice, copy and paste in the terminal  (in the choosed folder) and press ENTER:
```
chmod u+x setup.sh
```
#### 0.3 Start the setup
```
./setup.sh
```
     
#### 04. Run InfaScript.sh and ENJOY :D
```
cd infaScript
```
```
./Infascript.sh
```
